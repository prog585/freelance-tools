# upwork-tools

A collection of productivity tools for Upwork freelancers.

## Tools

### 🔍 [semantic-matcher](./semantic-matcher)
Semantic job matching using Qdrant vector DB + BGE embeddings.  
Embed your Upwork profile once, add job listings, get ranked matches by semantic similarity — no keyword rules.

**Stack:** Python · Qdrant · sentence-transformers (BGE-base-en-v1.5) · Click · Rich
